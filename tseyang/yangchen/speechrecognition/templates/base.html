speech recognition

Speech can be deﬁned as the expression of or the ability to express thoughts and feelings by articulate sounds. Speech has beenoneofthefundamentalmethodsofcommunicationforthe human race. Our brains have been designed to understand, interpret and generate trained languages. In recent years, the importanceofspeechincomputersisalsobeingrealized,asitsuse has opened wide realm of possibilities.Speech Recognition System is a system that is responsible for conversion of spoken audio input into a text, providing meaningfuloutput. Thissystemismadeupoflanguagemodels, prediction algorithms and digital signal processing algorithms. The output then could be used for a number of tasks ranging from a simple text conversion to the operation of a complete software package including the operating system itself. There has already been numerous implementations of Speech Recognition Systems for various languages in numerous domains. It is to be noted that there has also been some attemptsatNepaliSpeechRecognitionsystemdevelopment. But, no signiﬁcant traces of such implementations have been found. This project aims to further contribute to the ﬁeld. The project also aims to improve the accuracy of the system using modern hardware and algorithm optimizations.
